# 详解

CSS3动画功能

- transition  从一个属性值平滑的过度到另个属性值

transition: <过度属性名称> <过度时间> <过渡模式>

![](http://on891bjlf.bkt.clouddn.com/css3/5.png)

- animation  支持关键帧技术 在网页上完成更加复杂的动画效果
